Copsewood.Net LETS Accounting Training/Test Site Page

PyLETS CGI is a simple, lightweight on-line mono-LETS accounting system written in Python. It is designed for small transaction volumes where locked files provide a more cost effective data persistance mechanism than a full-blown database solution, which would be needed for higher transaction volumes. This page provides access to a test system kept , and the current download . For more information read the source code (provided under the GNU Public License Version 2).

Enter the test and training system. Data is non-permanent and will be periodically wiped. Please contact lets_account_training@copsewood.net to access this system for evaluation and to request reset of the administration (admin) account.
